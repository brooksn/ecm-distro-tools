# semv

Parse, test and compare semantic versions

### Examples

```sh
semv -test v1.1.x v1.1.1-rc1
```

```sh
semv -is-prerelease v1.1.1-rc1
```

```sh
semv -parse v1.1.1-rc1 -o 
```
